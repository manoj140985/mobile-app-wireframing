# Mobile App Wireframing

## Project Title
Expense Tracker Mobile Application

## Description
This project demonstrates the design of low-fidelity wireframes for an Expense Tracker mobile application. The wireframes were created using Figma and focus on the application's layout, navigation, and user flow before the final UI design.

## Objectives
- Conduct basic user research
- Create a user persona
- Design a user flow
- Develop low-fidelity wireframes
- Follow the design thinking process

## Wireframes Included
1. Login Screen
2. Dashboard
3. Add Expense
4. Reports

## Tool Used
- Figma

## Design Thinking Process
- Empathize
- Define
- Ideate
- Prototype
- Test
